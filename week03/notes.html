Objects

Creating objects

      const spiderman = {};
      const spiderman = new Object();

properties that are attributes of an object is a method. Methods can be called using the dot notaion of accessing properties.

      superman.fly()

You can use a for...in loop to iterate over all the properties of an object

      for(const key in superman) {
        console.log(key + ": " + superman[key]);
      }
      for(const key of Object.keys(superman)) {
        console.log(key);
      }

You can use the delete keyword in order to remove properties from an onject

      delete obj.property;

=============OBJECTS ARE COPIED BY REFERENCE!=============

When passing an object as a param to a function, you can destructure that bad boy!

      function greet({greeting,name,age}) {
        return `${greeting}! My name is ${name} and I am ${age} years old.`;
      }

You can use the this keyword in order to access properties of an object from within a method.

Objects can be used to namespace functions and variables.

JSON object can be used to marshal and parse other objects to and from strings

Regex objects can be created using a literal or the constructor

      const pattern = /[a-zA-Z]+ing$/;
      const pattern = new RegExp('[a-zA-Z]+ing');

Document Object Model

"The Document Object Model, or DOM for short, represents an HTML document as a network of connected nodes that form a tree-like structure."

You can get an element by its ID

      const h1 = document.getElementById('title');

Using document.querySelector will find the first element based off of the css selectors.

You can programmatically get and set the attributes of an element:

      wonderWoman.setAttribute('class', 'villain');
      wonderWoman.getAttribute('class');

=============using className can overwrite all other class names!=============

using the classList API is much better for updating classes

      wonderWoman.classList.add('warrior');
      wonderWoman.classList.remove('warrior');
      wonderWoman.classList.toggle('hero');
      wonderWoman.classList.contains('hero');

Vanilla uses removeChild to remove a node from the document and replaceChild in order to replace one

innerHTML is how you can set the children and values directly for a node

When a script tag is added to a node using innerHTML will not run so that anything malicious will not get executed.

Events

Event handlers are functions that get called when the user performs a certain action

      document.body.addEventListener("click", doSomething);

Listeners can be added a couple of ways

  • inline on the element
  • using dot notation of a node
  • using addEventListener

event.target can help determine which element an event is coming from. You can also use the coordinates using screenX and screenY

event types:

  • mouse events
  • keyboard events
  • touch events

You need to consider event propogation when handling events. They can either be captured or allowed to bubble. Using stopPropogation can prevent other handlers from firing correctly.

event delegation is attaching a listener to a parent in order to capture events from the children. Useful for attaching listeners to the document for dynamic content so only have to add the listener once.
