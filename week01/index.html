Mobile First

  • Building for large screens first is like trying to fit marsh mellows into a piggy bank
  • What is responsive retrofitting?
  • Small devices: average of two devices per person per household

Understanding Users

Steve Jobs's rules:

  1. Be there
  2. Be useful
  3. Be quick

Reachability: How to do users hold their device? What is easy to touch and what is hard?

Speed is huge. Networks are the biggest determiner of speed. We can optimize in order to help with speed:

  1. reduce image sizes
  2. minify resource files
  3. gzip files

Strategic Menu Design

Moving menus to the bottom can make a huge difference. increased reachability, higher perceived speed, engagement, and satisfaction.

"Menu" made to look like a button is the best performing style.

Friendly Forms

Ways to improve forms:

  1. put labels inside
  2. make taller
  3. use float labels
  4. minimize data collected/shown
  5. pagination with progress form
  6. enable autocomplete
  7. allow show password
  8. use the right keyboard
